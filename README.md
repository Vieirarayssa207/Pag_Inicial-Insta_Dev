# Recriação da Página Inicial do Instagram (Exercício Educacional)

Este projeto consiste em uma recriação da página inicial de login do Instagram, desenvolvida como um exercício educacional no âmbito do projeto "Dev em Dobro". O objetivo principal é demonstrar habilidades em HTML e CSS, além de servir como portfólio para avaliação do meu progresso como desenvolvedor.

## Disclaimer

**IMPORTANTE:** Esta página é uma recriação não oficial para fins de estudo e demonstração de habilidades.

*   Não é o site oficial do Instagram ou Meta.
*   Não sou afiliado(a) ao Instagram ou Meta de forma alguma.
*   Este projeto é estritamente para fins educacionais.

## Funcionalidades

*   **Recriação Visual:** A página replica o design da antiga tela de login do Instagram.
*   **Código Autoral:** Todo o código HTML e CSS foi escrito integralmente por mim.
*   **Foco Educacional:** O projeto serve como um exercício prático e demonstração de habilidades.

**Observação:** Este projeto não possui funcionalidades de login, não coleta dados de usuários e não tem qualquer associação comercial com o Instagram/Meta.

## Tecnologias Utilizadas

*   HTML
*   CSS

## Termos do GitHub

Este projeto está em conformidade com os Termos do GitHub para Produtos e Recursos Adicionais, que permitem cópias de sites para aprendizado, desde que:

*   O código seja original (de minha autoria).
*   Não haja coleta de dados.
*   Não se passe pelo site oficial.
